# CodeMaze-API
CodeMaze - RESTful API that follows ONION Architecture Implementation
Supports:
- JSON, XML, CSV Content-type.
- PATCH Operation.
- Paging, Filtering, Sorting, Searching, Data Sharding.
- HATEOAS.
- OPTIONS, HEADS Request.
- Versioning APIs.
- Caching, Rate Limit.
- Identity Framework and JWT.
